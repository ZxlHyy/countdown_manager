## 0.1.0

- Initial public release.
- Added `Countdown` for high precision countdown timers.
- Added `Countup` for forward timers with optional `maxDuration`.
- Added `TimerController` with `pause`, `resume`, `stop`, and `reset`.
- Added declarative `paused` support.
- Added lifecycle resume refresh for background correction.
- Added low-power second-level scheduler based on monotonic elapsed realtime.
- Added custom rendering through `timerWidgetBuilder`.
